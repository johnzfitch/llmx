### Data Flow

```
User (Claude Code)
    ↓ MCP stdio
LlmxServer (mcp_server.rs)
    ↓ Arc<Mutex<IndexStore>>
IndexStore (storage.rs)
    ├─→ Disk: ~/.llmx/indexes/{id}.json
    └─→ Cache: HashMap<String, IndexFile>
        ↓
    IndexFile (model.rs)
        ├─ files: Vec<FileMeta>
        ├─ chunks: Vec<Chunk>
        ├─ inverted_index: BTreeMap<String, PostingList>
        └─ stats: IndexStats
```

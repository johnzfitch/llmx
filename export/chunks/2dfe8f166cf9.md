### Combining Multiple Searches

For complex queries, use multiple searches:

```typescript
// 1. Broad semantic search
{
  "query": "authentication patterns",
  "use_semantic": true,
  "limit": 20
}

// 2. Narrow to specific implementation
{
  "query": "JWT verify",
  "use_semantic": false,
  "filters": {
    "path_prefix": "src/auth/"
  },
  "limit": 5
}
```

### Async Considerations
Browser embedding is async (WebGPU kernel dispatch). Structure code accordingly:

```rust
// Don't block main thread
let embeddings = spawn_local(async {
    let embedder = Embedder::new().await?;
    embedder.embed_batch(chunks).await
});
```

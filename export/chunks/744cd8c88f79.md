### Why Burn?
- **Rust-native**: Single toolchain, compiles to WASM
- **ONNX import**: Converts model to Rust code at build time (not runtime interpretation)
- **WGPU backend**: WebGPU in browser, Vulkan/Metal native
- **No JS interop**: Pure Rust/WASM stack

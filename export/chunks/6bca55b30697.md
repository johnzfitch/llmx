### Example 1: Conceptual Search

**Query**: "authentication logic"

**BM25 only** (default):
```json
{
  "query": "authentication logic",
  "use_semantic": false
}
```
Returns: Exact matches for "authentication" OR "logic"

**Hybrid semantic**:
```json
{
  "query": "authentication logic",
  "use_semantic": true
}
```
Returns: Auth logic + login flows + user verification + session management

### Runtime Errors

**"WebGPU embeddings not yet implemented"**
- Normal - model code not yet generated on first build
- Build process generates code in `src/model/`
- Uncomment model loading code after first successful build

**"Using hash-based embeddings (models unavailable)"**
- WebGPU initialization failed
- CPU backend initialization failed
- Falls back to hash-based (Phase 5 compatible)
- Check browser WebGPU support

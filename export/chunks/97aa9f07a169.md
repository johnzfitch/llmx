### Problem: Old indexes don't have embeddings

**Cause**: Index created before Phase 5.

**Solution**: Re-index the codebase:
```json
{
  "tool": "llmx_index",
  "arguments": {
    "paths": ["/path/to/project"]
  }
}
```

New index will include embeddings automatically.

### Progressive Enhancement Strategy

Don't block on embeddings. Quality filtering works immediately:

```typescript
// Phase 1: Immediate (quality only)
const filtered = chunks.filter(c => c.qualityScore >= 0.6);
let searchFn = (q) => bm25Search(q, filtered);

// Phase 2: Background embedding
const embedder = new Embedder();
embedder.init().then(async () => {
  const embeddings = await embedder.embedBatch(filtered.map(c => c.content));
  filtered.forEach((c, i) => c.embedding = embeddings[i]);
  
  // Upgrade search
  searchFn = (q) => hybridSearch(q, filtered, embedder);
  showToast("âœ¨ Semantic search enabled");
});
```

## Search API

- `search(query, filters) -> [chunk_id]` (BM25-like scoring)
- `get_chunk(chunk_id) -> chunk text + metadata`
- `list_outline(path) -> headings`
- `list_symbols(path) -> symbols`

Filters:

- `path_prefix`
- `kind`
- `heading_prefix`
- `symbol_prefix`

### Performance Issues

**Slow first search**
- WebGPU shader compilation on first use
- Add warmup call after embedder initialization
- Subsequent searches should be fast

**Embeddings take >100ms each**
- Use batch processing instead of individual embeds
- WebGPU overhead amortizes over batches
- Target: <500ms for 32 embeddings

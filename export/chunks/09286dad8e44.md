### From Phase 5 (Hash-based)

**No breaking changes.** Phase 5 indexes continue to work:
- Hash-based embeddings still supported
- Linear combination still available
- Search API unchanged

**To upgrade:**
1. Build with Phase 6 code
2. Re-generate embeddings with Burn model
3. Set `embedding_model = "bge-small-en-v1.5"`
4. Searches will use RRF by default

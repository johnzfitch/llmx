### ONNX → Burn Conversion (Build Script)

**build.rs:**
```rust
use burn_import::onnx::ModelGen;

fn main() {
    // Convert ONNX to Burn model at compile time
    ModelGen::new()
        .input("models/bge-small-en-v1.5.onnx")
        .out_dir("src/model/")
        .run_from_script();
}
```

This generates Rust code in `src/model/` that you import directly. No runtime ONNX parsing.

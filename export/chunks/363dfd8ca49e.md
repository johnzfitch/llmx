### Basic Embedding Generation

```javascript
// Initialize embedder
const embedder = await new Embedder();

// Single text
const embedding = embedder.embed("Hello world");
console.log(embedding); // Float32Array(384)

// Batch processing
const texts = ["text 1", "text 2", "text 3"];
const embeddings = embedder.embedBatch(texts);
console.log(embeddings); // Array of Float32Array(384)

// Check backend
console.log(embedder.modelId()); // "bge-small-en-v1.5"
console.log(embedder.dimension()); // 384
```

### First Build

On first build, `build.rs` will:
1. Download bge-small-en-v1.5.onnx (~33MB)
2. Download tokenizer.json (~500KB)
3. Convert ONNX to Burn Rust code (may take a minute)

Subsequent builds reuse cached models.

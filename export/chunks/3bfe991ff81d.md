### Project-Specific Config

Create `.llmx.toml` in project root:

```toml
[index]
chunk_size = 300
include = ["src/**", "lib/**"]
exclude = ["**/*_test.rs", "**/*.generated.*"]

[symbols]
extract = ["function", "class", "struct", "trait", "interface"]

[output]
format = "markdown"  # or "json"
```

### How It Fits

llm.cat already has quality scoring (Layer E from LLMCAT_INTEGRATION_PLAN.md) - pure JS math, instant. Embeddings add semantic search on top:

```
Upload → Chunk → Quality Score (JS, instant)
                      ↓
              Embed (WebGPU, async background) → Index with embeddings
                                                       ↓
                                         Search: BM25 + Semantic + Quality boost
```

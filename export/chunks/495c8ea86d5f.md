### SearchInput Compatibility

Phase 5 code continues to work:

```json
{
  "use_semantic": true
}
```

Phase 6 code can specify strategy:

```json
{
  "use_semantic": true,
  "hybrid_strategy": "rrf"
}
```

Default strategy is RRF if not specified.

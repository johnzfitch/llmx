### Search (New optional parameter)

```bash
# BM25-only search (default, backward compatible)
{
  "tool": "llmx_search",
  "arguments": {
    "index_id": "abc123",
    "query": "authentication logic"
  }
}

# Hybrid semantic search (new in Phase 5)
{
  "tool": "llmx_search",
  "arguments": {
    "index_id": "abc123",
    "query": "authentication logic",
    "use_semantic": true
  }
}
```

### ⚠️ Model Management (Browser-Specific)
- [ ] **No filesystem**: Models stored in IndexedDB, NOT `~/.llmx/models/`
- [ ] **Fetch weights**: Use browser `fetch()` API, not `ureq`/`reqwest`
- [ ] **CDN hosting**: Host `bge-small-en-v1.5` weights on CDN with CORS headers
- [ ] **Cache strategy**: Implement IndexedDB get/set for weights persistence
- [ ] **Offline fallback**: If fetch fails AND no cache → fall back to hash-based (Phase 5)

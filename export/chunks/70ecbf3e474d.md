### Build Commands

```bash
# Development build
cargo build --features mcp

# Release build
cargo build --release --features mcp --bin llmx-mcp

# Run tests
cargo test

# Check code quality
cargo clippy --all-features

# Generate documentation
cargo doc --open --features mcp

# Run benchmarks
cargo bench --bench baseline
```

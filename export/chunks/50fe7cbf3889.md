### Exploring Unfamiliar Codebases

Use semantic search for discovery:

```json
{
  "query": "how does caching work",
  "use_semantic": true,
  "limit": 15
}
```

Then refine with BM25:

```json
{
  "query": "CacheManager class",
  "use_semantic": false,
  "limit": 5
}
```

### Performance Targets (Browser/WebGPU)
| Operation | Target | Notes |
|-----------|--------|-------|
| Model init | <3s | First load, cached after |
| Single embedding | <100ms | WebGPU kernel launch overhead |
| Batch (32) | <500ms | Amortizes overhead |
| Hybrid search | <300ms | Acceptable for interactive |

WebGPU has higher per-call overhead than native GPU. Batch when possible.

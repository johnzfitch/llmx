### Build Time (build.rs)

```rust
1. Download bge-small-en-v1.5.onnx from HuggingFace (if not cached)
2. Download tokenizer.json from HuggingFace (if not cached)
3. Convert ONNX to Burn Rust code via burn-import
4. Generate model code in src/model/ directory
```

**Note:** Models are gitignored - fresh clones auto-fetch on first build.
